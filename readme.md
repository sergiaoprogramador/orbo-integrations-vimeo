# Teste de Integrações e Uso do Vimeo

### Introduction
Do desafio proposto:

- Criar e Integrar o Front-end do Vimeo SDK
    - Chat de Bate-Papo
    - Ferramentas de Enquetes

- Criar uma Classe PHP  para usar e testar os métodos HTTP de utilização do Vimeo no Backend
    - CRUD Videos files
    - CRUD Livestream
    - Dados RMTP
